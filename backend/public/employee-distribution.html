Live package contents

Every ZIP includes the current monitor, installer, requirements, and backend URL. Windows ships with install.bat; macOS and Linux ship with install.sh and a macOS double-click wrapper. If Python or Tesseract are missing, the installer will try to install them automatically using the platform package manager.

Note: the ZIPs are the supported direct-download path for employee rollout. They reflect the live Render deployment, so a fresh download always pulls the latest build.

Expected Setup Flow

  1. Download the ZIP that matches the employee's OS.
  2. Run the included installer for that platform.
  3. Complete the browser setup page if prompted.
  4. Save the employee profile.
  5. The monitor starts and stays tied to the live backend.

Expected Setup Flow

  1. Download the ZIP above.
  2. Extract the ZIP.
  3. Run the launcher inside:
    • Windows: double-click deploy_automated.bat
    • macOS: double-click install.command
    • Linux: run install.sh in a terminal
  4. Complete the browser setup page when it opens.
  5. Save the employee profile — monitor starts automatically.

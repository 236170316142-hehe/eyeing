Live package contents

ZIPs are generated live from the Render deployment. A fresh download always includes the latest monitor code and backend URL.

Expected Setup Flow

  1. Download the ZIP for the employee's OS, or use a Direct Installer button.
  2. Run the included launcher: deploy_automated.bat (Windows), install.command (macOS), or install.sh (Linux). Enterprise Windows: install_enterprise.bat.
  3. Complete the browser setup page if prompted.
  4. Save the employee profile.
  5. The monitor starts and stays tied to the live backend.

Windows bootstrap (PowerShell)

Run this on a target PC to download and auto-install:

powershell -ExecutionPolicy Bypass -Command "iwr -UseBasicParsing {{ORIGIN}}/api/employee/bootstrap.ps1 -OutFile $env:TEMP\employee-bootstrap.ps1; & $env:TEMP\employee-bootstrap.ps1"

Setup URL

Direct setup page:

{{ORIGIN}}/setup.html?autoclose=1

If macOS blocks install.command

Open Terminal in the extracted folder and run:

xattr -dr com.apple.quarantine . chmod +x install.sh install.command ./install.sh

One-command install for employee PCs

Run this in PowerShell on the employee machine:

powershell -ExecutionPolicy Bypass -Command "iwr -UseBasicParsing {{ORIGIN}}/api/employee/bootstrap.ps1 -OutFile $env:TEMP\employee-bootstrap.ps1; & $env:TEMP\employee-bootstrap.ps1"
Note: Browser sandbox rules do not allow direct extraction or .bat execution from a webpage click. The provided PowerShell installer handles those steps reliably.

Expected Setup Flow

  1. Auto installer downloads and extracts the package.
  2. install.bat is executed twice.
  3. Employee setup page opens with automation flags.
  4. Employee clicks Save.
  5. Page attempts to close automatically.
  6. Monitor launch is requested automatically.

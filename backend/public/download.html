python verify_tesseract.py python verify_autostart.py

🍺 The installer calls brew install tesseract automatically. If Homebrew is not installed, step 3 below shows how to get it first.

Step-by-step installation — macOS

1
Download the ZIP above Click the blue button. macOS saves it to your Downloads folder.
2
Extract & open Terminal Double-click the ZIP to extract it. Then open Terminal (Spotlight → Terminal).
cd ~/Downloads/eyeing-mac
3
Install Homebrew (if not already installed) Skip this step if brew --version already works.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
After installing on Apple Silicon (M1/M2/M3), also run:
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)"
4
Ensure Python 3.8+ is available macOS ships Python 3 since Ventura. To check:
python3 --version
If missing: brew install python
5
Run the installer This installs Tesseract via Homebrew, all Python packages, and sets up autostart via LaunchAgent.
python3 install_and_run.py --autostart
6
Complete the web setup form A browser tab will open at /setup.html. Fill in your employee details and click Save Setup.
7
Allow permissions when prompted macOS may ask for Screen Recording and Accessibility permissions. Grant both in System Settings → Privacy & Security for the monitor to function.

📦 The installer auto-detects your package manager (apt-get, dnf, or yum) and installs Tesseract OCR without any manual steps.

Step-by-step installation — Linux

1
Download the ZIP above Click the blue button or use the terminal:
wget "https://eyeing.onrender.com/api/employee/linux.zip" -O eyeing-linux.zip
2
Extract the ZIP
unzip linux.zip -d ~/eyeing cd ~/eyeing
3
Ensure Python 3.8+ is installed
# Ubuntu / Debian sudo apt-get install -y python3 python3-pip unzip # Fedora / RHEL sudo dnf install -y python3 python3-pip unzip
4
Run the installer The script auto-installs Tesseract OCR and all Python dependencies, then registers a systemd user service and desktop autostart entry.
python3 install_and_run.py --autostart
The installer will call sudo apt-get install tesseract-ocr (or the equivalent for your distro). Enter your password when prompted.
5
Complete the web setup form A browser tab opens at /setup.html. Fill in your employee details and click Save Setup.
6
Confirm the service is running
systemctl --user status employee-monitor.service
The monitor starts on login automatically. Restarting the PC confirms autostart works.
7
Headless / server installs (optional) If the machine has no desktop session, register the system-wide service instead:
cd linux bash setup_systemd.sh
